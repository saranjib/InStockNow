# ShelfSync – Real-Time Inventory Tracking System

## 📌 Overview
ShelfSync helps customers check real-time product availability across nearby stores and avoid wasted shopping trips.

---

## 🚀 Features
- User Authentication
- Product Search
- Store Locator
- Real-Time Inventory Updates
- Product Reservation System
- Admin Dashboard
- Google Maps Integration

---

## 🏗️ Tech Stack
Frontend: React.js  
Backend: Flask (Python)  
Database: PostgreSQL  
Realtime: Socket.IO  

---
